1. Click on "Start" then click on Command Prompt.
  2. To list all of the directories in Command Prompt, type in 'dir'
  3. To make a directory type 'md'.
  4. To make a file type 'copy NUL (ProgramName)'
  5. To edit a file type 'notepad (ProgramName)'
  6. To choose a directory press 'cd'. You can also leave the directories by pressing 'cd..'
